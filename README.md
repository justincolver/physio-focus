# The Physio Focus

Simple static website for The Physio Focus, built with plain HTML and CSS.

## Project structure

```text
.
├── index.html
├── styles.css
└── images/
    ├── favicon.png
    ├── hero.png
    ├── kaz.png
    └── logo.png
```

## Local editing

1. Open `index.html` in a browser to preview the site.
2. Edit page content in `index.html`.
3. Edit visual styles in `styles.css`.
4. Replace the placeholder image paths by adding the final files into `images/`.
5. Replace any PNG assets in `images/` if you want to swap in final exports later.

## GitHub / Vercel

This project is ready to deploy as a static site on GitHub Pages or Vercel with no build step.

- GitHub Pages: push the folder contents to a repository and serve from the root.
- Vercel: import the repository and deploy it as a static site with the default settings.

## Files to add manually before pushing

- `images/logo.png`
- `images/hero.png`
- `images/kaz.png`
- `images/favicon.png`
